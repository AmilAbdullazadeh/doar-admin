# ApexCharts Charts

<p class="sbdocs-desc">
    React-ApexCharts is a wrapper component for ApexCharts ready to be
    integrated into your react.js application to create stunning React Charts.
</p>

## Bar Chart

ApexCharts supports bars and any combinations of these, in the same plot and even on the same data series.

<Preview withToolbar>
    <Story name="Bar Chart">
        <ApexBarChart
            options={ApexBarData.options}
            series={ApexBarData.series}
            height="350px"
            width="100%"
        />
    </Story>
</Preview>

## Stacked Chart

With this plugin, you can have stack the series. This is useful if you wish to display both a total and the constituents it is made of.

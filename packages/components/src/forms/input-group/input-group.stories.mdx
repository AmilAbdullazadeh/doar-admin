# Input Group

<p className="sbdocs-desc">
    Easily extend form controls by adding text, buttons, or button groups on
    either side of textual inputs, custom selects, and custom file inputs.
</p>

## Basic Example

Place one add-on or button on either side of an input. You may also place one on both sides of an input.

<Preview withToolbar>
    <Story name="Basic Example">
        <InputGroup>
            <InputGroupAddon dir="prepend">
                <InputGroupText>@</InputGroupText>
            </InputGroupAddon>
            <Input
                type="text"
                id="username"
                name="username"
                placeholder="Username"
            />
        </InputGroup>
        <InputGroup>
            <Input
                type="text"
                id="recipent-username"
                name="recipent-username"
                placeholder="Recipent's Username"
            />
            <InputGroupAddon>
                <InputGroupText>@example.com</InputGroupText>
            </InputGroupAddon>
        </InputGroup>
        <InputGroup>
            <InputGroupAddon dir="prepend">
                <InputGroupText>$</InputGroupText>
            </InputGroupAddon>
            <Input
                type="text"
                id="recipent-username"
                name="recipent-username"
                placeholder="Recipent's Username"
            />
            <InputGroupAddon>
                <InputGroupText>.00</InputGroupText>
            </InputGroupAddon>
        </InputGroup>
    </Story>
</Preview>

## Checkboxes and Radios

Place any checkbox or radio option within an input group’s addon instead of text.

<Preview withToolbar>
    <Story name="Checkboxes and Radios">
        <InputGroup>
            <InputGroupAddon dir="prepend">
                <InputGroupText>
                    <Checkbox name="checkbox" id="checkbox" pl="15px" />
                </InputGroupText>
            </InputGroupAddon>
            <Input type="text" id="username" name="username" />
        </InputGroup>
        <InputGroup>
            <InputGroupAddon dir="prepend">
                <InputGroupText>
                    <Radio name="radio" id="radio" pl="15px" />
                </InputGroupText>
            </InputGroupAddon>
            <Input
                type="text"
                id="recipent-username"
                name="recipent-username"
            />
        </InputGroup>
    </Story>
</Preview>

## Multiple Inputs

While multiple inputs are supported visually, validation styles are only available for input groups with a single input.

<Preview withToolbar>
    <Story name="Multiple Inputs">
        <InputGroup>
            <InputGroupAddon dir="prepend">
                <InputGroupText>Fullname</InputGroupText>
            </InputGroupAddon>
            <Input
                type="text"
                id="f_name"
                name="f_name"
                placeholder="First Name"
            />
            <Input
                type="text"
                id="l_name"
                name="l_name"
                placeholder="Last Name"
            />
        </InputGroup>
    </Story>
</Preview>

## Multiple Addons

Multiple add-ons are supported and can be mixed with checkbox and radio input versions.

<Preview withToolbar>
    <Story name="Multiple Addons">
        <InputGroup>
            <InputGroupAddon dir="prepend">
                <InputGroupText>$.</InputGroupText>
                <InputGroupText>00</InputGroupText>
            </InputGroupAddon>
            <Input type="text" id="f_name" name="f_name" />
        </InputGroup>
        <InputGroup>
            <Input type="text" id="f_name" name="f_name" />
            <InputGroupAddon>
                <InputGroupText>$.</InputGroupText>
                <InputGroupText>00</InputGroupText>
            </InputGroupAddon>
        </InputGroup>
    </Story>
</Preview>

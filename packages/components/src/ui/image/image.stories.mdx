## Responsive Images

Images in Bootstrap are made responsive with `fluid`. max-width: 100%; and height: auto; are applied to the image so that it scales with the parent element.

<Preview withToolbar>
    <Story name="Responsive Images">
        <Image src={exmaplImage} alt="Responsive Images" fluid />
    </Story>
</Preview>

## Image Thumbnails

In addition to our border-radius utilities, you can use `thumbnail` to give an image a rounded 1px border appearance.

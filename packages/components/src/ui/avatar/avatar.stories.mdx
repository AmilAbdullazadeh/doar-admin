# Overview

Avatars are user's face image that you can use throughout the design from tables to dialog or menus with options for sizes and shapes.

<Canvas>
    <Story
        name="Avatar Controls"
        argTypes={{
            size: {
                control: {
                    type: `select`,
                    options: ["default", "xs", "sm", "md", "lg", "xl", "xxl"],
                },
            },
            shape: {
                control: {
                    type: `select`,
                    options: ["circle", "rounded", "square"],
                },
            },
            status: {
                control: {
                    type: `inline-radio`,
                    options: ["online", "offline"],
                },
            },
            clasName: {
                control: {
                    type: `text`,
                },
            },
        }}
        args={{
            size: `default`,
            shape: `circle`,
        }}
    >
        {(args) => {
            const { ...restArgs } = args;
            return (
                <Avatar {...restArgs}>
                    <img src={image} />
                </Avatar>
            );
        }}
    </Story>
</Canvas>

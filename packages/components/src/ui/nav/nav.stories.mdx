# Navs

<p class="sbdocs-desc">
    Documentation and examples for how to use navigation components.
</p>

## Basic Example

The base `Nav` component to the active and disabled states. Swap modifier props to switch between each style.

<Preview withToolbar>
    <Story name="Basic Example">
        <Nav>
            <NavLink path="#!" active>
                Home
            </NavLink>
            <NavLink path="#!">About</NavLink>
            <NavLink path="#!">Contact</NavLink>
        </Nav>
    </Story>
</Preview>

## Horizontal Alignment

Change the horizontal alignment of your nav with `align` props. By default, navs are left-aligned, but you can easily change them to center or right aligned.

<Preview withToolbar>
    <Story name="Horizontal Alignment">
        <Nav align="center">
            <NavLink path="#!" active>
                Home
            </NavLink>
            <NavLink path="#!">About</NavLink>
            <NavLink path="#!">Contact</NavLink>
        </Nav>
        <Nav align="right">
            <NavLink path="#!" active>
                Home
            </NavLink>
            <NavLink path="#!">About</NavLink>
            <NavLink path="#!">Contact</NavLink>
        </Nav>
    </Story>
</Preview>

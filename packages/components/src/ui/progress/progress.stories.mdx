## Basic Example

Progress components are built with two HTML elements, some CSS to set the width, and a few attributes.

<Preview withToolbar>
    <Story name="Basic Example">
        <Progress />
    </Story>
</Preview>

## Labels

Add labels to your progress bars by plassing `isLabel`.

<Preview withToolbar>
    <Story name="Labels">
        <Progress height="20px" isLabel />
    </Story>
</Preview>

## Height

We only set a height value on the progress, so if you change that value the inner progress bar will automatically resize accordingly.

<Preview withToolbar>
    <Story name="Height">
        <Progress height="20px" now={60} />
        <Progress height="5px" now={30} />
    </Story>
</Preview>

## Background

Use background utility classes to change the appearance of individual progress bars.

<Preview withToolbar>
    <Story name="Background">
        <Progress bg="success" now={80} />
        <Progress bg="info" now={60} />
        <Progress bg="#dc3545" now={90} />
    </Story>
</Preview>

## Striped

Pass `striped` to any progress bar to apply a stripe via CSS gradient over the progress barâ€™s background color.

<Preview withToolbar>
    <Story name="Striped">
        <Progress striped bg="success" now={80} />
        <Progress striped bg="info" now={60} />
        <Progress striped bg="#dc3545" now={90} />
    </Story>
</Preview>

## Animated Stripes

The striped gradient can also be animated.
